# The Idea Vault

A static landing page for a personal idea collection.

## Project structure

- `index.html` — the main page
- `styles.css` — shared styles extracted from the inline stylesheet
- `script.js` — interactive behavior extracted from the inline script

## Usage

Open `index.html` in a browser to preview the site.

## Notes

- Add or update ideas in `script.js` inside the `ideas` array.
- Add essays in `script.js` inside the `essays` array.
